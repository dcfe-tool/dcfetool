About DCFE Tool

In the process of data analysis, it is observed that 60% of the time is spent in cleaning and preprocessing the data. To overcome this drawback and to reduce the time spent in preprocessing we have developed a tool named Data Curation and Feature Engineering Tool (DCFE Tool). Epidemiologists, Researchers or any kind of Data analysts can use this web based application tool to upload, explore and analyze the epidemic/pandemic or any health care or other domain datasets.

The tool provides the provision for the user to upload datasets which are in .csv or .xlsx file format. This is a DST funded project with duration of 3 years. Project was sanctioned on Aprilâ€™2018 and completed on Jan, 2022. Project sanction number DST/ICPS/EDA/2018/SL.No:18.

This tool was developed in CUDA lab, Department of Computer Applications, NIT Trichy under the Principal Investigator Dr.S.Nickolas, Professor, Department of Computer Applications, and Co-Principal Investigator Dr.S.Mary Saira Banu, Professor and Head, Department of Computer Science and Engineering, NIT, Trichy.

The tool is developed using Model Template Controller (MTC) in Django Web Based Framework with supporting coding using python, Html5, CSS3, Bootstrap4, JavaScript and Jquery.
