The Reason we developed DCFE Tool

Big data has been described as large, dynamic, or highly heterogeneous data that requires new forms of processing to enable enhanced decision making and insight discovery. One such field of study is Epidemiology of Healthcare data in which there is vast availability of big data that are left unexplored and preprocessed.

Many of these new forms of processing can be described as data curation, that is, the care of data to ensure that it maintains its value over time. This can be achieved by coupling the data imputation and feature engineering techniques with the data exploration tools which will help to enrich the data quality further.

We find that, the more data we need to curate for analytics and research purpose, the more costly and complex curation becomes, mostly domain experts or data scientists are not scalable. Hence, our motive is to focus on automation of data curation process using machine driven, human guided techniques to enable data curation on scale.

Primary Focus

Our primary focus is on Data Curation tool in- order to make the process of Obtaining, Exploring, Cleaning, Transforming and Visualizing data much more accessible to huge number of beneficiaries including Researchers, Data scientists, Epidemiologists, Health care professionals and various other data analysts and decision makers.

In this regard we have developed the Data Curation and Feature Engineering (DCFE) tool with Data Imputation Feature Engineering as its key features along with Data Visualization. The tool is accessible through online and users can readily use it by uploading datasets with file extension .csv or .xlsx files.
